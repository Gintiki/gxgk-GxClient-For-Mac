## 摘要

**ALevoClient**, 一个Mac版 联想天宫802.1x认证客户端，适用与才用改网络的校园网办公网等场所登录使用，项目基于开源*LevoClient*开发，并针对Mac做了改进并增加了GUI、断网重连、网卡选择等功能.

> 大学即将毕业了，总想着给即将逝去的大学时光留下点什么，然后就花了几天时间做了这个，希望能给大学中向我一样忍受了三年间接性断网困扰的人带来点希望~
